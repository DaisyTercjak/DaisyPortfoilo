Project Path

IDE Used

For this project, I used WebStorm as the IDE for JavaScript development, providing powerful tools for efficient coding and debugging. For Python and Flask, I utilized PyCharm, which offered robust features for backend development, including code assistance, debugging, and project management.

Project Plan

To organize the project effectively, I utilized Jira and Confluence. Jira was used to manage and track tasks, enhancing visibility and accountability, while Confluence served as a collaborative space for planning and documentation. Together, these tools streamlined task execution and ensured a well-structured project plan.

Version Control

For version control, I used GitHub, creating a new branch for each task from Jira. This approach ensured organized development, allowing precise control over how the project was built and facilitating effective collaboration and code management.
